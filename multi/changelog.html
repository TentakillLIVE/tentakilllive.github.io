Changelog

A full history of updates to EightEyes Multi.

v2.1.1 1 March 2026

Updated Overlay style for stream titles.

UI

The "Overlay" option for stream titles as been updated.
  • Before: Semi-transparent black bar behind the title text.
  • Now: Bar has been removed. Title text now has a stroke outline around the text.
v2.1.0 28 February 2026

Made some changes to the site that are not related directly to the multistream functionality.

  • Implemented an official privacy policy as required by the Chrome Web Store for review and verification of the Chromium version of our helper extension
  • Migrated away from Google Fonts to Bunny Fonts
  • Added a proper changelog
  • Updated Help Section.
  • Both versions of our helper extension are now Google and Mozilla verified.
v2.0.0 23 February 2026

Please see the below changes. Many were implemented specifically to help improve the performance of the website.

UI

  • Updated Settings Panel to a more compact pop out overlay.
  • Added Play button that plays all streams on the current tab. 
Note: It is recommended that you allow the page to fully finish loading all streams before using the Play feature, due to performance limitations with the embed Twitch player.

Settings

  • Added setting to adjust the size of titles.
  • Added setting to toggle between the original title style ("Top"), and a new style that overlays the titles over the streams ("Overlay").
  • Added setting to adjust the positioning of titles. "Top" and "Bottom" are self explanatory. "Top Avoid" and "Bottom Avoid" are similar. However, they avoid the top and bottom of the screen respectively to help streamers avoid their stream overlays from blocking titles.
  • Added setting to disable animations.
  • Adding setting to toggle between the original stream grid layout (now referred to as "Padded"), and a new style that disables padding and rounded corners to make full use of the screen real estate (referred to as "Full")
  • Added setting to toggle chat. Chat is now disabled by default. When disabled, chat is not loaded into memory at all.

HOTKEYS

  • Added hotkeys (1-9) to switch directly between the first 9 tabs.
  • Added hotkey (V) to open and close the top pane.
  • Added hotkey (C)  to open and close chat when enabled.
  • Added hotkey (T) to toggle stream titles.

OTHER

Made several behind-the-scenes changes aimed at optimizing the site, and improving compatibility for Gecko-based browsers.
v1.1.0 21 February 2026

New features.

Hotkeys

  • Press T to mute all streams
  • Cycle through tabs with A and D

Titles

Added a toggle to enable stream titles to appear above each stream. This makes it very easy to identify which stream is which.

Helper  extension

This is the biggest addition of this update. We now have a helper extension which enables Twitch Turbo users to be able to authenticate their account in the Twitch streams, allowing them to view the streams without ads.
v1.0.0 10 February 2026

Official release.

  • Support for both Twitch and Youtube
  • Support for Twitch and YT chat
  • Seamlessly rearrange streams using drag-and-drop
  • Organize streams into tab groups
  • Rename both streams and tabs to better identify them
  • Focus Video button to zoom in on a stream while keeping the others on the side
  • Focus Audio to quickly unmute a specific stream while muting others.
