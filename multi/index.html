Twitch Turbo & Ads

Browsers block third-party cookies by default, so embedded Twitch players can't verify your Turbo subscription and serve ads anyway. We built a companion browser extension to fix this.

EightEyes Turbo Helper

Reads your Twitch auth token directly and injects it into embedded player requests — Twitch sees your Turbo subscription and skips ads entirely.

Chromium Firefox-based
  1. Download the zip and extract it — you can review all source files before installing if you wish.
  2. Open chrome://extensions in your Chromium-based browser.
  3. Enable Developer Mode using the toggle in the top-right corner.
  4. Click Load unpacked and select the extracted extension folder.
  5. The EightEyes icon will appear in your toolbar. Open it and confirm the green active status.
Alternative Chromium browsers: If your browser supports CRX installation, you can simply drag and drop the .crx file directly onto the Extensions page instead.
  1. Download the .xpi file.
  2. Open the menu → Add-ons and Themes → gear icon → Install Add-on From File…
  3. Select the .xpi file you downloaded.
  4. Confirm the installation prompt. The EightEyes icon will appear in your toolbar.
Want to review the source? A full source zip is available to inspect before or after installation.

Your privacy is fully respected. The extension runs entirely on your computer — your auth token is read locally, never transmitted to any server, and only ever used to authenticate directly with Twitch's own servers.
