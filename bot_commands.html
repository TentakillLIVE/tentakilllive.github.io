Payouts [Unrestricted]
Calculates the payouts based on buy-in and participants for races. Uses a 60%/30%/10% split.
Calculate
Format
!payouts [buy-in] [participants]
Example
!payouts 1000 10
Output
6000, 3000, 1000
Timer [VIP+]
Creates a timer for a set duration that returns a message in chat after the duration.
Set a Timer
Format
!timer [duration] [message]
The duration can be a number representing seconds. For example, setting a Timer with 20 seconds:
!timer 20 Hey I am shown after 20 seconds
The duration can also be not only a number, for example setting a Timer for 1 hour 20 minutes:
!timer 1h20m Hey I am shown after 1 hour and 20 minutes
Parts of the duration can be omitted. For example:
!timer 3h13m Hey I am shown after 3 hours and 13 minutes
!timer 13m30s Hey I am shown after 13 minutes and 30 seconds
• hours: h
• minutes: m
• seconds: s
Show Timers
Shows all currently running timers and their remaining time.
!timers
Clear Timers
Deletes all currently running timers.
!cleartimers
Stopwatch [Moderators+]
Each user has their own stopwatch. Each user can only have 1 stopwatch active at a time.
Start the stopwatch
!sw start
Stop the stopwatch
!sw stop
Show stopwatch split
!sw split
Reminders [Moderators+]
Each channel has his own reminders, listed by number depending on the creation time, note that they work like the timers: Reminder 2 can be reminder 3 if reminder 1 gets deleted
Add a reminder
Format
!addreminder [message]
Example
!addreminder Do the laundry
List reminders
Lists all reminders, and its associated number, which can be used to delete the reminder.
!reminders
Delete a reminder
Deletes a specific reminder using its associated number.
Format
!delreminder [reminder number]
Example
!delreminder 1
Spotify Control [Moderators+]
To control Spotify, a device must be active. It controls the currently active device of the given account.
If nothing is playing for a while, it is not possible to control Spotify over chat.
Pause playback
!pause
Resume playback
!play
Skip current song
!next
Go to previous song
!previous
Play a song
Plays a given song. The song can be a Spotify link, or just a string that gets looked up and plays the first match.
The song will be played immediately
Format
!play [song/song link]
Examples
!play https://open.spotify.com/track/4PTG3Z6ehGkBFwjybzWkR8?si=036d1d341b564552
!play Darude - Sandstorm
Check volume
!volume
Set volume
Set the volume, from 0 - 100.
Format
!volume [0 - 100]
Examples
!volume 35
!volume 100
